# Parser_WB_eshop
Парсер создан для сбора информации по категории и состоит из 3-х независимых этапов
Сперва собираются все ссылки со страниц(get_link) и сохраняются в csv
Затем в get_data из файла csv со ссылками получаем первичные данные и записываем в новый csv(после получения можем отфильтровать по цене и бренду,удалив лишнее)
Финальный этап происходит в create_final_table, где мы берем csv файл из п4 и делаем более удобный вариант таблицы выводя каждую характеристику в заголовки.
